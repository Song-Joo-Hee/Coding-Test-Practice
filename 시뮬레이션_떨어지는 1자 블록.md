```python
"""
문제. 코드트리 intermediate low
0과 1로만 채워져 있는 n * n 크기의 격자판 정보가 주어진다.
0은 빈 칸을, 1은 해당 칸에 블럭이 채워져 있음을 뜻한다.
이때 1 * m 크기의 블럭이 격자판 위에서 떨어진다.
이 블럭은 k번째 열 부터 k + m - 1번째 열까지의 공간을 차지하며,
가장 위에서부터 밑으로 떨어진다.
만약 이 블럭이 떨어지는 도중 단 한곳에라도 이미 격자판 위에 놓여있던 블럭과
맞닿게 된다거나, 혹은 바닥에 닿게 된다면 떨어지는 것을 멈추게 된다.

격자판의 정보와 떨어질 블럭의 정보가 주어졌을 때, 블럭이 떨어진 이후의 상태를 출력하는 프로그램을 작성해라.

입력.
첫번째 줄 : 격자의 크기를 나타내는 n과 블록의 크기를 나타내는 m, 그리고 블록이 떨어질 위치에 대한 정보를 나타내는 k
(1 * m 크기의 블록이 k번째 열 부터 k + m - 1번째 열까지의 공간을 차지)
두번째 줄 : 각 행에 해당하는 숫자 (0 또는 1)

출력.
블럭이 떨어진 이후의 격자판 상태를 출력

아이디어.
<시뮬레이션>
1) 초기 값
2) 한 턴의 움직임
3) 전체 움직임

--> 해당 행이 비어있는지 확인 하는 함수
--> 그 전 행이 비어있다면 자신을 return하는 함수
--> return된 행을 1로 바꿔주는 작업
"""

n, m, k = map(int, input().split())
grid = [list(map(int, input().split())) for _ in range(n)]

def all_blank(row, col_s, col_e):
    return all([
        not grid[row][col]
        for col in range(col_s, col_e + 1)
    ])

def get_target_row():
    for row in range(n-1):
        if not all_blank(row+1, k, k+m-1):
            return row

    return n-1

k -= 1

target_row = get_target_row()

for col in range(k, k+m):
    grid[target_row][col] = 1

for i in grid:
    print(*i)
```
