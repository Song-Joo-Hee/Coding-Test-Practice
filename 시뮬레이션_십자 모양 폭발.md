```python
"""
문제. 
1이상 100이하의 숫자로 구성된 n * n 크기의 격자판이 주어집니다. 
이때 특정 위치를 선택하면, 그 위치를 중심으로 십자 모양으로 폭탄이 터지게 됩니다. 
십자 모양의 크기는 선택된 위치에 적혀있는 숫자로 정해지며, 터진 이후에는 중력에 의해 숫자들이 아래로 떨어지게 됩니다.
십자 모양의 크기는 선택된 숫자에 비례하여 커집니다. 
선택된 숫자가 1인 경우에는 자기 자신만 터지게 되고, 
선택한 숫자가 2인 경우에는 자신을 포함하여 인접한 4개의 격자 역시 터지게 되며, 
3인 경우에는 자신을 제외한 상하좌우 방향으로 각각 2개씩이 더 터지게 됩니다. 
숫자가 4 이상인 경우에도 마찬가지의 규칙에 따라 해당 범위만큼 터지게 됩니다.

폭탄이 터져야 하는 범위가 격자 판을 벗어나게 되더라도, 격자 안에서만 폭탄이 터지게 됩니다.

입력. 
첫 번째 줄: 격자의 크기를 나타내는 n이 주어집니다.
두 번째 줄: n개의 줄에 걸쳐 각 행에 해당하는 n개의 숫자가 공백을 사이에 두고 주어집니다.
그 다음 줄 : 폭탄이 터질 중심 위치의 정보를 나타내는 (r, c) 값이 공백을 사이에 두고 주어집니다. 이는 r행 c열이 중심 위치임을 의미합니다. 

출력.
폭탄이 터지고 나서 중력이 작용한 뒤의 결과를 출력합니다.

n개의 줄에 걸쳐 각 행에 해당하는 n개의 숫자를 공백을 사이에 두고 출력합니다. 
만약 해당 위치에 아무 숫자도 적혀있지 않은 경우라면 0을 출력합니다.
"""


n = int(input())
grid = [
    list(map(int, input().split()))
    for _ in range(n)
]
next_grid = [
    [0 for _ in range(n)]
    for _ in range(n)
]


def in_bomb_range(x, y, center_x, center_y, bomb_range):
    return (x == center_x or y == center_y) and \
           abs(x - center_x) + abs(y - center_y) < bomb_range


def bomb(center_x, center_y):
    bomb_range = grid[center_x][center_y]
    
    # 1. 폭탄이 터질 위치는 0으로 채우기
    for i in range(n):
        for j in range(n):
            if in_bomb_range(i, j, center_x, center_y, bomb_range):
                grid[i][j] = 0
	
    # 2. 폭탄이 터진 이후의 결과를 next_grid에 저장
    for j in range(n):
        next_row = n - 1
        for i in range(n - 1, -1, -1):
            if grid[i][j]:
                next_grid[next_row][j] = grid[i][j]
                next_row -= 1
                
    # 3. grid로 다시 값을 옮겨준다.
    for i in range(n):
        for j in range(n):
            grid[i][j] = next_grid[i][j]

            
r, c = tuple(map(int, input().split()))
bomb(r - 1, c - 1)

for i in range(n):
    for j in range(n):
        print(grid[i][j], end=" ")
    print()
```
