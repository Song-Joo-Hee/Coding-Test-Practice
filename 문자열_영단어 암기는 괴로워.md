### 문제
* 백준 20920번
* https://www.acmicpc.net/problem/20920

### 풀이
* dict 사용
* 입력을 받으면서 dict에 넣기
* 그 후, lambda를 이용하여 dict를 정렬해주기

```python
import sys
input = sys.stdin.readline

n,m = map(int, input().split())
note = dict()

for _ in range(n):
    word = input().rstrip()
    if len(word) >= 4:
        if word in note:
            note[word] += 1
        else:
            note[word] = 1

note = sorted(note.items(), key=lambda x: (-x[1], -len(x[0]), x[0]))

for i in note:
    print(i[0])

```
