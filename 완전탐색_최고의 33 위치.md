```python
"""
문제. 코드트리 intermediate low
N * N 크기의 격자 정보가 주어집니다. 이때 해당 위치에 동전이 있다면 1, 없다면 0이 주어집니다.
N * N 격자를 벗어나지 않도록 3 * 3 크기의 격자를 적절하게 잘 잡아서
해당 범위 안에 들어있는 동전의 개수를 최대로 하는 프로그램을 작성해보세요.

입력.
첫째 줄 : 격자의 크기 n
두번째 줄 : n개의 줄에 걸쳐 격자에 대한 정보(0 또는 1)
3 <= n < 20

출력.
최대 동전의 수
"""


n = int(input())
grid = [list(map(int, input().split())) for _ in range(n)]

max_gold = 0
# 모든 격자를 다 돌아야 하니까 이중 for문 만들기
for row in range(n-2): #3*3 격자를 탐색해야하니까 n*n 격자를 벗어나지 않는 범위로 설정
    for col in range(n-2):
    #(row, col)은 기준점이 된다.
        num_of_gold = 0
        for i in range(row, row+3):
            for j in range(col, col+3):
            # 기준점을 처음으로 3*3을 돈다.
                num_of_gold += grid[i][j]
    max_gold = max(max_gold, num_of_gold)

```
